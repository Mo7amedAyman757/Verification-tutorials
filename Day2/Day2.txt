*Constraint Exercise 2 slide 137:*
### First File: Package (my_pkg)  
1. Create a package named my_pkg.  
2. Declare three parameters:  
   - HEIGHT = 10  
   - WIDTH = 10  
   - PERCENT_WHITE = 20  
3. Define an enumerated type colors_t with two values:  
   - BLACK  
   - WHITE  
4. Create a class named screen:  
   - Declare a 2D randomized unpacked array pixels of type colors_t, with dimensions [HEIGHT][WIDTH].  
   - Define a constraint block (colors_c) to enforce a distribution:  
     - WHITE should appear PERCENT_WHITE% of the time.  
     - BLACK should appear (100 - PERCENT_WHITE)% of the time.  
   - Implement a print_screen function (void):  
     - Iterates over pixels.  
     - Prints each row of the array.  

---

### Second File: Module (image)

3. Use an initial block to:
